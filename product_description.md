# 3.5 Preliminary Product Description

The **Vista Fin AI** (Financial Integration & Support System) is designed to facilitate seamless personal financial management, combining real-time investment tracking, AI-driven insights, and automated expense categorization in one platform. Key features include:

1.  **Secure Authentication Module:** Robust login/signup functionality using Firebase Authentication, ensuring secure access to personal financial data.
2.  **Financial Dashboard Module:** A centralized hub displaying real-time net worth, recent transactions, and portfolio summaries with interactive charts.
3.  **Investment Tracking Module:** Supports real-time tracking of stocks and market data via Alpha Vantage API, allowing users to monitor portfolio performance live.
4.  **AI Financial Advisor (Chat) Module:** An integrated chatbot powered by Google Gemini AI that provides personalized financial advice, investment strategies, and answers to financial queries.
5.  **Smart Expense Tracking Module:** Automatically categorizes income and expenses using AI algorithms, visualizing spending habits through dynamic graphs.
6.  **Price Alert & Notification Module:** A proactive system that monitors stock prices in the background and sends real-time push notifications (FCM) when target prices or profit/loss thresholds are met.

The product ensures data security, real-time accuracy, and user-centric design, enabling users to make informed financial decisions without relying on disparate tools. It optimizes the wealth management workflow by integrating essential financial functionalities into a single, intuitive platform.
